Hey! Let's Code

Hey! Let’s Code is an online two week long class for children between the ages of 7 and 15 to learn the foundation of programming principles in a fun and collaborative way. We teach our students the fundamentals of Scratch or Web Development throughout a week long class taught by highly qualified counselors who have real world programming experience.

We are now offering 3 types of classes: Scratch, Web Development, and Roblox Game Design
